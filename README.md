# STM32 Project Collection

这个仓库目前收录了两个基于 STM32 的工程，分别用于外设驱动/综合实验，以及循迹小车基础控制。

## 仓库结构

```text
.
├─ 2024.8.30_1
├─ 循迹基本代码
└─ README.md
```

## 项目说明

### 1. `2024.8.30_1`

这是一个偏综合实验性质的 STM32 工程，包含较多外设驱动和功能模块，适合作为功能验证、课程设计或模块开发的代码基础。

已包含的部分模块有：

- AS608 指纹识别
- ESP8266
- RC522 RFID
- BH1750 光照传感器
- DHT11 温湿度
- MAX30102
- 超声波模块
- 步进电机
- 触摸相关模块
- 语音相关模块
- 点阵显示
- 图形界面相关代码

该工程中还能看到 `task`、`Quest`、`UI` 等目录，说明它不仅有底层驱动，还包含更高层的实验任务和界面逻辑。

### 2. `循迹基本代码`

这是一个 STM32 循迹小车基础工程，主要实现红外循迹、按键启动、前进/转向修正/停止等基本控制逻辑。

核心逻辑位于：

- `循迹基本代码/User/main.c`
- `循迹基本代码/Hardware/Irtracking.c`
- `循迹基本代码/Hardware/robot.c`
- `循迹基本代码/Hardware/PWM.c`

主程序流程大致为：

1. 初始化机器人驱动
2. 初始化按键
3. 初始化循迹红外模块
4. 等待按键启动
5. 根据左右红外状态执行前进、左转修正、右转修正或刹车

## 开发环境

这两个工程都更适合使用以下环境打开和编译：

- Keil uVision
- STM32 标准外设库工程环境

其中 `循迹基本代码` 目录中包含：

- `Project.uvprojx`
- `Project.uvoptx`

可直接作为 Keil 工程入口文件使用。

## 说明

仓库已添加 `.gitignore`，用于忽略常见的编译产物和本机配置文件，例如：

- `Objects/`
- `Listings/`
- `*.uvguix.*`
- `DebugConfig/`
- `.DS_Store`

这样可以让仓库内容更聚焦在源码和工程文件本身。
